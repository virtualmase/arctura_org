QR

Scanning the QR code: Open any Base-compatible wallet — Coinbase Wallet, MetaMask, Rainbow, or Trust Wallet — and select Send → Scan QR. The code encodes ethereum:0x...@8453, the EIP-681 standard URI. Wallets with Base support auto-select the correct network on scan. One scan. Enter your amount. Confirm the transmission. ENS names (arcturiancouncil.base.eth) resolve in wallets with ENS support.
