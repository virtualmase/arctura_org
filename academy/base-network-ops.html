Transmission · 03 · Prerequisites

Who This Is For

  • Arctura operators who want to move beyond watching the chain to running it
  • Builders ready to deploy real on-chain infrastructure, not testnet toys
  • Anyone managing treasury, credentials, or agent funding at the network level
  • Developers building products that touch Base — wallets, contracts, NFTs

What You Need

  • Basic JavaScript — you'll call contracts from code
  • A Base-compatible wallet already set up (Coinbase Wallet recommended)
  • Small amount of ETH on Base for gas (covered in Module 2)
  • No prior smart contract experience required — we start from chain primitives
